# Inventory Management System API

## Overview
The Inventory Management System API is a backend application built using Django Rest Framework (DRF) that provides a secure, efficient way to manage inventory items. It supports CRUD operations, JWT-based authentication, Redis caching, and comprehensive logging and testing.

## Table of Contents
    User registration and authentication using JWT tokens.
    CRUD operations for items in the inventory.
    Token-based authorization for secure access.
    Caching mechanisms to improve performance .
    Detailed logging for monitoring and debugging.
